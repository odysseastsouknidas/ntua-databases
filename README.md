# Databases
### ECE NTUA - Flow L: Computer Software - 6th Semester
### Academic Year 2023–24

This repository contains our solutions for the semester project of the [Databases course](https://www.ece.ntua.gr/en/undergraduate/courses/3123). This course covers the basics of database systems - from ER Modeling, Relational Algebra and SQL, to more advanced topics such as indexing, query processing and optimization. The semester project involves designing and implementing a database for a cooking competition, including recipes, ingredients, chefs, and competition episodes.
